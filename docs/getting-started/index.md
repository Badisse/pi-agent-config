# Getting Started

This section covers installing the Pi agent config and setting up a new project.

## Prerequisites

- [Pi coding agent](https://github.com/mariozechner/pi-coding-agent) installed globally
- [GitHub CLI (`gh`)](https://cli.github.com/) authenticated
- [Docker](https://www.docker.com/) (for the Ralph sandbox loop)
- [tmux](https://github.com/tmux/tmux) (for background Ralph sessions)
- Git configured with SSH keys

## Overview

1. **[:material-download: Install](installation.md)** — Clone the config repo to `~/.pi/agent`
2. **[:material-folder-open: Project setup](project-setup.md)** — Initialize any project with `/ralph init`
