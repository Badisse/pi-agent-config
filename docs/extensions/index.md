# Extensions

Pi extensions add slash commands, security guards, and workflow automation. They live at `~/.pi/agent/extensions/` and are loaded automatically.

| Extension | What it does |
|---|---|
| [Ralph](ralph.md) | `/ralph` — manage autonomous agent loops |
| [Git Workflow](git-workflow.md) | `/branch`, `/commit`, `/checkpoint`, `/pr-summary` |
| [Security Gate](security-gate.md) | Blocks destructive and suspicious commands |
| [Protected Paths](protected-paths.md) | Blocks access to sensitive files |
| [Pi Docs](pi-docs.md) | Generates HTML documentation from codebase |
