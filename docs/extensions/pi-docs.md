# Pi Docs Extension

!!! info "Source"
    `~/.pi/agent/extensions/pi-docs/`

Generates HTML documentation from the codebase.

## Files

| File | Purpose |
|---|---|
| `index.ts` | Extension entry point |
| `data-collector.ts` | Collects codebase data for docs |
| `html-template.ts` | HTML template for generated docs |

This extension is a utility for generating project documentation summaries in HTML format.
