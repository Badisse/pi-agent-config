# Workflows

End-to-end session patterns for common development scenarios. Each workflow shows the exact skills and commands to run.

| Workflow | Duration | Autonomous? | Best for |
|---|---|---|---|
| [Full Feature](full-feature.md) | 1–2 hours | Partially | New features from idea to deployed code |
| [Quick Bug Fix](quick-bugfix.md) | 15–30 min | No | Diagnosing and fixing a bug |
| [Architecture Day](architecture-day.md) | 2–4 hours | Partially | Refactoring, deepening modules |
| [Fire and Forget](fire-and-forget.md) | Variable | Fully | Multiple parallel autonomous agents |
